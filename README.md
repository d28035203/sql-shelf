# Library Card Chaos

A **SQLite** mini library system — DBMS-lab energy from **coursework-style practice**.

Issue books. Return books. Shame overdue patrons (politely).

```bash
python3 library.py init
python3 library.py add "Operating Systems" "Galvin"
python3 library.py issue 1 "student42"
python3 library.py list
python3 library.py return 1
```
